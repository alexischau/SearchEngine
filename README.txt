1. how to run the code that creates the index
- Install the following packages/libraries in the console
    pip install lxml
    pip install unidecode
    pip install Flask
    pip install bs4
    pip install nltk
- type either "python3 gui.py" or "python gui.py" in the console

2. how to start the search interface 
- Follow the instructions above
- The console should print out “Running on http://127.0.0.1:5000”
- Click on the url http://127.0.0.1:5000

3. how to perform a simple query
-  type a query in the search box and press ‘enter’ to submit. This will take you to a new page with the results
- click on the back button to return to the search page
